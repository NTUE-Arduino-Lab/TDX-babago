# The F2E - Take a Bus

## 作品說明

Week 3：全台公車動態時刻查詢應用服務</br>
使用 React 實作，串接政府公開 API。</br>
感謝 [Lilian Chen](https://2021.thef2e.com/users/6296432819610583955) 精美的 UI 設計！

## 系統說明

版本
- react - 17.0.2
- npm - 6.14.11

安裝 `npm install`

執行 `npm start`

排版
- Prettier

語法檢查
- ESLint - 檢查 js 語法
- Stylelint - 檢查 CSS 語法

## 資料夾說明

- asset - 使用圖片、檔案
- component - 頁面元件
- router - 頁面路由
- sidebar - 側邊欄的頁面
- store - 全域變數存取

## 使用技術

- React
- Axios
- Prettier
- ELint
- Stylelint

- git
